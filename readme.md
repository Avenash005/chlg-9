# Bug Tracker

## What It Is
A web app to log and track coding bugs and fixes.

## The Problem It Solves
I kept forgetting bugs I encountered and how I fixed them. This app stores them so I can reuse solutions instead of debugging again.

## What I Intentionally Excluded
- Authentication: Adds complexity (JWT/sessions) not needed for a single-user MVP.

## Tech Stack
- Backend: Node.js + Express + SQLite
- Frontend: Vanilla JS
- Deployment: Render + Netlify
